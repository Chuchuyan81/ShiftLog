# ?? втоматическое тестирование ShiftLog

## писание системы

омплексная система автоматического тестирования для приложения 'урнал абочих Смен'.

## айлы

- autotest.js - основной скрипт тестирования
- test-runner.html - интерфейс для запуска тестов
- TESTING_README.md - данная инструкция

## апуск тестов

1. ткройте test-runner.html в браузере
2. ажмите 'апустить все тесты'
3. нализируйте результаты в консоли

## то тестируется

- ? одключение к Supabase
- ? утентификация пользователей  
- ? CRUD операции (создание, чтение, обновление, удаление)
- ? RLS политики безопасности
- ? UI элементы и взаимодействия
- ? изнес-логика (расчеты, валидация)
- ? нтеграционные тесты
- ? роизводительность

## астройка

змените TEST_CONFIG в autotest.js для настройки тестов под вашу среду.
